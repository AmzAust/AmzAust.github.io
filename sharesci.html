The Homepage

Sharesci's landing page and main functionality. Here a user enters a term to search for an academic article. The search then uses an artifically intelligent algorithm to return the articles with the closest related content back to the user.


Search Results

This is what a user will see after they have entered their search term. A list of related articles is brought back in which they can click on to view a brief description or just download the article as a PDF. If there are more than 10 articles, we can use the pagination under the list to navigate to the next page.



User Profiles

The user profile page is where a user can go after creating their account. Here they can edit their account information. Accounts are also created so a user can upload articles in PDF format to the website and leave comments on an article page.



Uploads

The uploads page is where a user can use the drag and drop to upload one or multiple articles at the same time. These articles will be stored in the same database as all the other articles and can be returned in search results.



Comments

The comments section is located under each articles brief description. Users can upload a comment to any article they want after they have been logged in. The username and the comment body will be stored in a database separate from the articles.
